# React Bloom

## Project info

Simple demo storefront built with React, TypeScript and Tailwind CSS.

## How to run locally

```sh
# install
npm install

# start dev server
npm run dev
```

## How to deploy

Deploy to any static host (Vercel, Netlify, GitHub Pages, etc.). This repo contains a standard Vite + React build.

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui
